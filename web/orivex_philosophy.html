MISSION STATEMENT

私たちが信じること

身体を整えることは、自分を整えること。健康への意識が高まるとき、人はより前向きに、より誠実に、より豊かに生きられると私たちは信じています。

パーソナルトレーナーは、クライアントの身体だけでなく、その人の人生そのものに寄り添うプロフェッショナルです。ORIVEX.は、そのトレーナーひとりひとりをテクノロジーで支え、より多くの人を健康へと導く力を届けます。

「健康づくり人づくり」——これはORIVEX.が掲げる事業理念であり、私たちのすべての判断と行動の根幹にある言葉です。

OUR PILLARS

理念を支える3つの柱

Pillar 01 — Health

身体的健康を基点に、精神・社会的な豊かさまでを視野に入れたサポートを。

Pillar 02 — Growth

トレーナーが成長し続けられる環境こそが、クライアントの成長につながる。

Pillar 03 — Technology

テクノロジーは目的ではなく手段。人と人の関係を豊かにするために使う。

CORE VALUES

私たちの価値観

ORIVEX.のすべてのプロダクトと行動は、この価値観から生まれます。

VALUE — 01

クライアントファースト

トレーナーの成功は、クライアントの健康と成長によってのみ実現される。すべての提案と機能設計において、エンドユーザーの利益を最優先にします。

VALUE — 02

根拠ある提案

感覚や経験だけでなく、科学的エビデンスに基づいたサプリメント提案を。トレーナーの専門性を高め、クライアントとの信頼関係を育てます。

VALUE — 03

継続できる仕組み

一時的な結果より、長期的な習慣を。クライアントが健康を継続できるよう、トレーナーが伴走し続けられる環境をデザインします。

VALUE — 04

時間を価値に変える

提案・資料作成・共有にかかる時間をテクノロジーで短縮し、トレーナーが人に向き合う時間を最大化します。

VALUE — 05

誠実な関係性

トレーナーとクライアント、そしてORIVEX.との関係は、常に誠実であること。過剰な売り込みではなく、本質的な価値を届けます。

VALUE — 06

健康文化の醸成

個々の健康が、やがて社会全体の活力となる。ORIVEX.は、健康を当たり前の選択肢にするための文化づくりに貢献します。

日本のパーソナルトレーナー市場は拡大し続けていますが、多くのトレーナーが業務の煩雑さゆえに本来の指導に集中できていません。

ORIVEX.は、サプリメント提案の自動化という具体的な課題解決を通じて、トレーナーが「人と向き合う時間」を取り戻すことを支援します。

それは単なる業務効率化ではありません。トレーナーが余裕を持って一人ひとりのクライアントに向き合える環境こそが、健康づくりの質を高め、ひいては人づくりにつながると私たちは考えています。

健康な人が増えると、社会が変わる。ORIVEX.は、その変化の起点でありたいと思っています。
